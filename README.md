# Solana AI Tour - Web3 互动学习平台

![Solana AI Tour Banner](https://via.placeholder.com/1200x600/0c0a2a/9945FF?text=Solana+AI+Tour)

> **© 2026 Solana AI Tour · 为未来而建**

Solana AI Tour 是一个结合人工智能辅助的沉浸式 Solana 区块链学习平台。我们致力于通过游戏化的互动体验，降低 Web3 开发门槛，帮助开发者和初学者快速掌握 Solana 核心概念。

## 🚀 业务内容

本项目围绕 "AI + Solana" 的核心理念，构建了一条从零基础到部署应用的学习路径：

*   **Solana 基础**：理解高性能公链的基本原理，体验光速钱包创建。
*   **AI 辅助编程**：利用先进的 AI 模型，通过自然语言自动生成 Rust 智能合约代码。
*   **实战部署**：模拟真实的链上部署流程，让学习成果立即可见。

## ✨ 产品特色

### 1. 🎮 游戏化闯关体验
告别枯燥的文档阅读，我们设计了 **3 级闯关模式**：
*   **Level 1 - 光速钱包**：一键生成 Solana 钱包，体验极致速度。
*   **Level 2 - AI 代码生成**：输入 "创建一个代币转账程序"，AI 即刻为你编写代码。
*   **Level 3 - 链上部署**：亲手将智能合约部署到网络，获得即时反馈。

### 2. 🤖 沉浸式自动演示 (Auto-Demo)
独特的“观看演示”功能，提供全自动的操作回放。用户无需动手，即可完整预览从创建钱包到部署合约的全流程，所见即所得。

### 3. 🎨 极客风 UI 设计
*   **暗黑科技主题**：深紫色背景搭配霓虹渐变（Solana Green & Purple），营造未来感。
*   **流畅动画**：基于 `Framer Motion` 的粒子特效、平滑过渡和打字机效果。
*   **响应式布局**：完美适配桌面端与移动端体验。

## 🛠️ 技术栈

*   **前端框架**: React 18 + TypeScript + Vite
*   **样式库**: Tailwind CSS
*   **动画库**: Framer Motion
*   **图标库**: Lucide React

## 📦 安装与运行

确保您的环境已安装 Node.js (推荐 v16+)。

### 1. 克隆项目
```bash
git clone <repository-url>
cd solana-tour
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```
启动后访问 `http://localhost:5173` 即可体验。

### 4. 构建生产版本
```bash
npm run build
```
